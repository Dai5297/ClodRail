# 仿12306铁路微服务系统API接口文档

## 1. 接口设计规范

### 1.1 基础约定

**请求基础URL**: `http://localhost:8080/api`

**HTTP状态码**:
- 200: 请求成功
- 400: 请求参数错误
- 401: 未授权
- 403: 权限不足
- 404: 资源不存在
- 500: 服务器内部错误

**统一响应格式**:
```json
{
  "code": 200,
  "message": "操作成功",
  "data": {},
  "timestamp": 1703123456789
}
```

**分页响应格式**:
```json
{
  "code": 200,
  "message": "查询成功",
  "data": {
    "list": [],
    "total": 100,
    "page": 1,
    "size": 20,
    "totalPages": 5
  },
  "timestamp": 1703123456789
}
```

### 1.2 请求头规范

| 请求头名称 | 类型 | 必填 | 描述 |
|-----------|------|------|------|
| Content-Type | string | 是 | application/json |
| Authorization | string | 否 | Bearer {token} |
| Client-Type | string | 是 | web/mobile/admin |
| Version | string | 否 | API版本号 |

## 2. 用户认证模块

### 2.1 用户注册

**接口地址**: `POST /user/register`

**请求参数**:
```json
{
  "username": "user001",
  "password": "123456",
  "phone": "13800138000",
  "email": "user@example.com",
  "name": "张三",
  "idCard": "110101199001011234",
  "captcha": "1234"
}
```

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| username | string | 是 | 用户名，4-20位 |
| password | string | 是 | 密码，6-20位 |
| phone | string | 是 | 手机号 |
| email | string | 否 | 邮箱地址 |
| name | string | 是 | 真实姓名 |
| idCard | string | 是 | 身份证号 |
| captcha | string | 是 | 验证码 |

**响应数据**:
```json
{
  "code": 200,
  "message": "注册成功",
  "data": {
    "userId": 1,
    "username": "user001",
    "phone": "13800138000",
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "expiresIn": 7200
  }
}
```

## 11. 错误码说明

| 错误码 | HTTP状态码 | 描述 | 解决方案 |
|--------|-----------|------|----------|
| 1001 | 400 | 参数格式错误 | 检查请求参数格式 |
| 1002 | 400 | 必填参数缺失 | 补充必填参数 |
| 1003 | 400 | 参数值超出范围 | 检查参数取值范围 |
| 2001 | 401 | 未登录 | 重新登录获取token |
| 2002 | 401 | token已过期 | 刷新token或重新登录 |
| 2003 | 401 | token无效 | 重新登录 |
| 3001 | 403 | 权限不足 | 联系管理员分配权限 |
| 3002 | 403 | 账号被禁用 | 联系管理员解封 |
| 4001 | 404 | 用户不存在 | 检查用户ID |
| 4002 | 404 | 车次不存在 | 检查车次信息 |
| 4003 | 404 | 订单不存在 | 检查订单号 |
| 5001 | 500 | 数据库连接失败 | 联系技术支持 |
| 5002 | 500 | 第三方服务异常 | 稍后重试或联系技术支持 |
| 6001 | 400 | 用户名已存在 | 更换用户名 |
| 6002 | 400 | 手机号已注册 | 使用其他手机号或找回密码 |
| 6003 | 400 | 验证码错误 | 重新获取验证码 |
| 6004 | 400 | 密码格式不正确 | 密码需6-20位 |
| 7001 | 400 | 车票已售完 | 选择其他车次或座位类型 |
| 7002 | 400 | 余票不足 | 减少购票数量 |
| 7003 | 400 | 超出购票限制 | 检查购票规则 |
| 8001 | 400 | 支付金额不匹配 | 重新计算订单金额 |
| 8002 | 400 | 支付方式不支持 | 选择其他支付方式 |
| 8003 | 400 | 支付超时 | 重新发起支付 |
