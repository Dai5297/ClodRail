# 📚 ClodRail 项目文档

本目录按模块组织项目文档，每个文件夹对应一个服务模块。

## 📁 目录结构

```
Docs/
├── README.md                    # 本文件
├── 00-项目概述/                  # 项目整体介绍、架构设计
├── 01-用户服务/                  # rs-user 用户认证与权限
├── 02-车票服务/                  # rs-ticket 车站、线路、车票
├── 03-订单服务/                  # rs-order 订单、支付
├── 04-积分商城/                  # rs-mall 积分、商品
├── 05-客服助手/                  # rs-assistant AI客服
├── 06-网关服务/                  # rs-gateway 路由、鉴权
└── 99-部署运维/                  # 配置示例、部署指南
```

## 🖥️ 前端系统

本项目包含两个前端应用：

| 系统 | 目录 | 说明 |
|------|------|------|
| 用户端 | `RailwaySystem-Frontend/rs-user-web` | 面向乘客的购票系统 |
| 管理端 | `RailwaySystem-Frontend/rs-admin-web` | 后台管理系统 |

详见 [RailwaySystem-Frontend/README.md](../RailwaySystem-Frontend/README.md)

## 📋 服务端口速查

| 服务 | 端口 | 说明 |
|------|------|------|
| rs-gateway | 18080 | 统一网关入口 |
| rs-user | 18081 | 用户服务 |
| rs-ticket | 18082 | 车票服务 |
| rs-order | 18083 | 订单服务 |
| rs-assistant | 18084 | 客服服务 HTTP |
| Netty WS | 18085 | 客服 WebSocket |
| rs-mall | 18086 | 商城服务 |

2. 详细步骤见 [99-部署运维/快速启动.md](99-部署运维/快速启动.md)
