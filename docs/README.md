# ClodRail 文档导航

这个目录是项目的文档入口。为了方便开源读者快速定位内容，建议把它理解成三层：

1. `00-项目概述` 负责回答“这是什么项目、为什么这样设计”
2. `01` 到 `06` 负责回答“每个模块具体做了什么”
3. `99-部署运维` 负责回答“怎么跑起来、怎么配置环境”

## 开源阅读建议

如果你的目标是把仓库作为开源项目对外展示，这个目录最值得继续补充的内容有：

- 模块时序图或业务流程图
- 系统截图和演示视频
- 常见问题排查文档
- Roadmap 或待办列表
