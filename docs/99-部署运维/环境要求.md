# 环境要求

## 开发环境

| 软件 | 版本 | 说明 |
|------|------|------|
| JDK | 17+ | 推荐 17 |
| Maven | 3.8+ | 构建工具 |
| Node.js | 18+ | 前端构建 |
| MySQL | 8.0+ | 数据库 |
| Redis | 7.0+ | 缓存 |
| RabbitMQ | 3.12+ | 消息队列 |
| Nacos | 2.2+ | 注册中心 |

## 可选组件

| 软件 | 版本 | 说明 |
|------|------|------|
| Elasticsearch | 8.x | 商城搜索 |
| Seata Server | 1.7+ | 分布式事务 |
| XXL-JOB Admin | 2.4+ | 任务调度 |

## IDE 推荐

- IntelliJ IDEA（推荐旗舰版）
- VS Code（前端开发）

## 工具推荐

| 工具 | 用途 |
|------|------|
| Postman / Apifox | API 测试 |
| Navicat / DataGrip | 数据库管理 |
| RedisInsight | Redis 管理 |
| Git | 版本控制 |
