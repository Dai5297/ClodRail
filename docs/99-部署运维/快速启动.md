# 快速启动指南

## 1. 环境准备

确保已安装以下服务：
- MySQL 8.0+
- Redis 6.0+
- RabbitMQ 3.8+
- Nacos 2.x
- Seata 1.6+（可选，分布式事务）
- Elasticsearch 8.x（可选，商品搜索）

## 2. 数据库初始化

### 2.1 创建 Nacos 数据库并导入配置

```bash
# 创建 nacos 数据库
mysql -u root -p -e "CREATE DATABASE nacos DEFAULT CHARACTER SET utf8mb4;"

# 导入 Nacos 配置（包含所有共享配置）
mysql -u root -p nacos < Docs/99-部署运维/nacos.sql
```

导入后，Nacos 将自动包含以下共享配置：
- `shared-mysql.yaml` - MySQL 数据源配置
- `shared-redis.yaml` - Redis 配置
- `shared-rabbitmq.yaml` - RabbitMQ 配置
- `shared-seata.yaml` - Seata 分布式事务配置
- `shared-knife4j.yaml` - API 文档配置
- `shared-alipay.yaml` - 支付宝支付配置
- `shared-langchain4j.yaml` - AI 大模型配置

### 2.2 创建业务数据库

```bash
# 创建各服务数据库
mysql -u root -p -e "
CREATE DATABASE rs_user DEFAULT CHARACTER SET utf8mb4;
CREATE DATABASE rs_ticket DEFAULT CHARACTER SET utf8mb4;
CREATE DATABASE rs_order DEFAULT CHARACTER SET utf8mb4;
CREATE DATABASE rs_mall DEFAULT CHARACTER SET utf8mb4;
CREATE DATABASE seata DEFAULT CHARACTER SET utf8mb4;
"
```

## 3. 启动基础设施

```bash
# 启动 Nacos（单机模式）
cd nacos/bin
./startup.sh -m standalone

# 启动 MySQL、Redis、RabbitMQ（Docker 或本地）
```

## 4. 配置本地环境

在各服务的 `bootstrap.yaml` 中配置 Nacos 连接和环境变量：

```yaml
spring:
  cloud:
    nacos:
      server-addr: localhost:8848
      config:
        namespace: 83a90920-a536-4767-b536-3e99c506b1a0

# 配置各服务所需的环境变量
rs:
  mysql:
    host: localhost
    port: 3306
    username: root
    password: your_password
    db-name: rs_user  # 各服务对应的数据库名
  redis:
    host: localhost
    port: 6379
    password: your_password
```

## 5. 编译后端

```bash
cd RailwaySystem-Backend
mvn clean install -DskipTests
```

## 6. 启动后端服务

按顺序启动：

```bash
# 1. 网关
cd rs-gateway && mvn spring-boot:run

# 2. 用户服务
cd rs-service/rs-user && mvn spring-boot:run

# 3. 车票服务
cd rs-service/rs-ticket && mvn spring-boot:run

# 4. 订单服务
cd rs-service/rs-order && mvn spring-boot:run

# 5. 商城服务
cd rs-service/rs-mall && mvn spring-boot:run

# 6. 客服服务
cd rs-service/rs-assistant && mvn spring-boot:run
```

## 7. 启动前端

```bash
# 用户端
cd RailwaySystem-Frontend/rs-user-web
npm install && npm run dev

# 管理端
cd RailwaySystem-Frontend/rs-admin-web
npm install && npm run dev
```
