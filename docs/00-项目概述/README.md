## 快速了解

ClodRail 是一个仿 12306 的铁路票务微服务系统，包含：

- 🧑‍💼 用户服务 - 注册、登录、权限管理
- 🎫 车票服务 - 车站、线路、车次、余票
- 📦 订单服务 - 下单、支付、退票
- 🛒 积分商城 - 积分、商品兑换
- 🤖 客服助手 - AI 问答、实时通信
- 🚪 统一网关 - 路由、鉴权、限流
