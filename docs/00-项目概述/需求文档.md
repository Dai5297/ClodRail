# 需求文档

## 产品概述

本项目是一个仿 12306 的铁路微服务系统，为用户提供火车票预订、行程管理、积分兑换等服务。

## 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 手机号注册 | 购票、查询、个人信息管理、积分使用 |
| VIP用户（待完成） | 积分升级 | 优先购票、专属客服、积分加成 |
| 管理员 | 内部分配 | 系统管理、数据统计、用户管理 |

## 功能模块

| 模块 | 功能 |
|------|------|
| 用户管理 | 注册登录、个人信息、实名认证 |
| 车次管理 | 车次查询、时刻表、座位分布 |
| 购票模块 | 票务查询、在线购票、选座 |
| 订单管理 | 订单创建、支付、退改签 |
| 支付模块 | 多种支付方式、退款处理 |
| 积分商城 | 积分获取、商品兑换 |
| AI助手 | 智能客服、行程推荐 |

## 页面设计

### 用户端页面

| 页面 | 功能 |
|------|------|
| 首页 | 搜索、热门线路推荐 |
| 车次查询 | 筛选、车次列表 |
| 选座购票 | 座位图、乘客信息 |
| 支付页面 | 支付方式选择 |
| 个人中心 | 订单、积分、个人信息 |
| 积分商城 | 商品列表、兑换 |
| AI助手 | 智能对话 |

### 管理端页面

| 页面 | 功能 |
|------|------|
| 数据概览 | 仪表盘 |
| 用户管理 | 用户列表、操作 |
| 车次管理 | 车次信息维护 |
| 订单管理 | 订单查询、处理 |
| 积分管理 | 商品、规则 |
| 系统设置 | 参数配置 |

## 核心流程

### 购票流程

```
首页搜索 → 车次列表 → 选座 → 填写乘客 → 确认订单 → 支付 → 出票
```

### 积分兑换流程

```
积分商城 → 商品详情 → 确认兑换 → 扣除积分 → 生成订单
```
